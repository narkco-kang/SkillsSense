## 🎯 產品路線圖

### Version 1.0 — MVP（當前階段）

**目標**：驗證核心價值主張 — 「AI 理解需求，自動推薦技能 + 教程」

#### 功能範圍
- [x] AI 意圖解析（自然語言 → 結構化需求）
- [x] 多來源並行搜尋（Hugging Face / GitHub / Awesome Lists）
- [x] 生動圖文教程自動生成
- [x] AI 組合生成新技能（當需求不存在時）
- [x] 需求拆分引導
- [ ] 用戶反饋機制
- [ ] 搜尋歷史記錄
- [ ] 黑暗模式優化

#### 發布標準
- Build 通過 ✅
- 單元測試覆蓋核心邏輯 > 80%
- API 響應時間 < 3s
- 至少 3 個來源穩定運行
- 上線後 48 小時內收集第一批用戶反饋

---

### Version 2.0 — 社區與增長

**目標**：建立用戶社區，啟動病毒式增長

#### 功能範圍
- [ ] 用戶帳戶系統（GitHub OAuth）
- [ ] 技能收藏 / 喜歡功能
- [ ] 用戶貢獻的技能審核流程
- [ ] 技能評分與評論
- [ ] 個人化推薦（基於收藏歷史）
- [ ] 技能技能之間的依賴關係圖
- [ ] 分享功能（生成技能卡片圖片）

#### 增長目標
- 100 付費用戶
- 500 月活用戶（MAU）
- 社群討論串 > 10 個

---

### Version 3.0 — 平台化

**目標**：從工具轉型為平台，開放 API

#### 功能範圍
- [ ] 開放 API（開發者接入）
- [ ] 技能市場（付費技能上架）
- [ ] 技能訂閱制
- [ ] 企業版（團隊協作、品牌定制）
- [ ] AI Agent 自動化和工作流

#### 商業目標
- MRR > $10,000
- 10+ 企業客戶
- 100+ 付費技能
